Detailed Description

Cross-platform mutex abstraction for thread synchronization.

Provides a unified interface for mutex operations on Windows and POSIX systems. On Windows, it uses CRITICAL_SECTION for exclusive locking. On POSIX systems, it uses pthread read-write locks with support for shared (read) and exclusive (write) locking.

Macros:

  • mutex_init(m): Initialize the mutex.
  • mutex_destroy(m): Destroy the mutex and release resources.
  • mutex_lock(m): Acquire exclusive (write) lock.
  • mutex_unlock(m): Release the lock.
  • mutex_lock_shared(m): Acquire shared (read) lock (equivalent to exclusive lock on Windows).

Usage:

mutex_init(&m);
mutex_lock(&m);
// critical section
